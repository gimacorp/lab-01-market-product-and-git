## Roles and responsibilities

### 1. Backend Engineer

Designs and implements business logic in microservices such as Cart, OMS, Inventory, and Payment.
Responsible for API design, data consistency, service performance, and integration with databases and external systems.

### 2. Mobile Engineer (iOS / Android)

Develops and maintains the customer-facing mobile application.
Ensures smooth UX, integrates backend APIs, handles app performance, and releases updates through app stores.

### 3. DevOps / Site Reliability Engineer (SRE)

Builds and maintains CI/CD pipelines, deployment automation, and monitoring.
Ensures system reliability, scalability, fault tolerance, and fast recovery during incidents.

### 4. Platform / Infrastructure Engineer

Manages shared infrastructure such as Kubernetes clusters, API gateways, Kafka, Redis, and Elasticsearch.
Focuses on scalability, security, observability, and internal platform tooling for service teams.

### 5. QA Engineer

Designs and executes automated and manual tests across services and user flows.
Ensures functional correctness, regression safety, and reliability of releases, especially for critical flows like checkout and payment.

## Common skills across roles

* Understanding of **microservice architectures**
* Experience with **REST / gRPC APIs**
* Knowledge of **distributed systems** and eventual consistency
* Familiarity with **cloud-native environments** (Kubernetes, containers)
* Experience with **databases** (SQL and NoSQL)
* Observability skills (logging, metrics, tracing)
* CI/CD and version control (Git)
* Security basics (authentication, authorization, TLS)
* Cross-team communication and documentation skills
